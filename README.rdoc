= message_div_helpers

== Summary

This is a Rails plugin to aid the programmer in creating and displaying message divs
(such as notices or errors) based on variables saved in flash, or instance variables.

== Example usage

  # controller
  class FooController < ApplicationController
    message_div_helpers_for :success, :image => 'icons/accept.png'
    message_div_helpers_for :error,   :image => 'icons/exclamation.png'
    
    def create
      @foo = Foo.new(params[:foo])
      if @foo.save
        flash[:success] = "Foo successfully saved."
        redirect_to :action => 'list'
      else
        flash[:error] = "Had trouble saving the foo."
        flash[:foo] = @foo
        redirect_to :action => 'new'
      end
    end
  end
  
  # view
  <%= success_div_unless_blank %>
  <%= error_div_unless_blank %>
  
  # will render... (if success)
  <div id="success"><img src="/images/icons/accept.png" />&nbsp;&nbsp;Foo
  successfully saved.</div>
  
  # will render... (if error)
  <div id="error"><img src="/images/icons/exclamation.png" />&nbsp;&nbsp;Had trouble
  saving the foo.</div>
